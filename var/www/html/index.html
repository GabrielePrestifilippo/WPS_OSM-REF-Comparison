<span style="font-weight:bold;">How to use the tool:</span>
<p>Road network datasets must (for REF) and can (for OSM) be uploaded as a compressed ESRI shapfile. Click the <b>Browse</b> button, select the <b>.zip</b> file from your system and adjust its color using the <b>Layer Color</b> selector. Enter also the reference system of the dataset.</p>
<p>The OSM road network dataset can be also retrieved (using the Overpass API) by clicking the <b>Retrieve OSM data</b> button. This will retrieve OSM data for the current map view; alternatively, hold down the shift key to select a rectangle on the map view and then click the <b>Selection Retrieve</b> button.</p>

<p>When at least two datasets have been uploaded, click the <b>Compare menu</b> button. This will open a menu where you can specify the buffer width for the comparison, the datasets to be used as REF and OSM and (optionally) the mask for clipping the previous datasets &#45; to be previously uploaded as a compressed shapefile. Then click the <b>Start comparison</b> button to run the algorithm and download some statistics on the similarity between the datasets in a PDF file.</p>
